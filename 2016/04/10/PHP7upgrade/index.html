PHP 7 업그레이드

PHP 7의 출시와 함께 pecl을 이용한 oci8 설치가 php를 7으로 버전 업그레이드가 필요하게 되었다.
하지만, 아직 yum package에는 php 7버전이 올라와있지 않으므로 기존의 php 버전을 삭제후 IUS를 이용해 설치하는 방법을 소개하겠다.

1) root 접속


2) PHP 7 버전은 아직 yum package 에 등록되지 않았기 때문에 IUS 저장소를 이용해야 한다.


3) 기존의 PHP (버전 5.x, 4.x…)을 삭제한다.


4) PHP 7.x 버전을 IUS 로부터 다운 및 설치한다.


###5) 아파치 서버를 재시작한다.


6) 서버가 작동 중인지 확인한다.

정상이라면 아래와 같이 Active: active (running)이 초록색으로 나타날 것이다.



7) PHP 버전을 확인한다. 아래와 같이 PHP 7.x.x 가 확인된다면 설치가 된 것이다.



8) 마지막으로 *Firefox 브라우저를 이용해 PHP정보를 확인한다.

/etc/www/html/info.php 파일을 만들고 아래의 내용을 입력한 후 :wq 로 저장한후 브라우저를 열어 확인해본다.

브라우저 주소란에 `https://localhost/info.php를 입력하고 버전 및 모듈을 확인한다.
